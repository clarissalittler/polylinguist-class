# Lesson 13: Sorting & Searching - Exercises

## Exercise 1: Implement Sorting (Easy-Medium)

## Exercise 2: Quick Sort (Medium)

## Exercise 3: Merge Sort (Medium)
Implement merge sort iteratively and recursively.

## Exercise 4: Binary Search Variants (Medium)
Implement: first occurrence, last occurrence, closest element.

## Exercise 5: Custom Comparators (Easy)
