# Lesson 16: Graphs - Exercises

## Exercise 1: Graph Representation (Easy)
Implement adjacency list and matrix representations.

## Exercise 2: DFS and BFS (Easy-Medium)
Implement depth-first and breadth-first search.

## Exercise 3: Shortest Path (Medium)
Implement Dijkstra's algorithm for weighted graphs.

## Exercise 4: Cycle Detection (Medium)
Detect cycles in directed and undirected graphs.

## Exercise 5: Minimum Spanning Tree (Hard)
Implement Prim's or Kruskal's algorithm for MST.
