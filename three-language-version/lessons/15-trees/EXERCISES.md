# Lesson 15: Trees - Exercises

## Exercise 1: BST Implementation (Easy-Medium)
Implement insert, search, delete for BST.

## Exercise 2: Tree Traversals (Easy)
Implement inorder, preorder, postorder, level-order.

## Exercise 3: Tree Properties (Medium)
Implement height, size, isBalanced, isBST.

## Exercise 4: Lowest Common Ancestor (Medium)
Find LCA of two nodes in BST and general binary tree.

## Exercise 5: Serialize/Deserialize (Hard)
Implement tree serialization to string and deserialization.
