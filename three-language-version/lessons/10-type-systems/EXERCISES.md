# Lesson 10: Type Systems - Exercises

## Exercise 1: Type Annotations (Easy)
Add type annotations to Python functions. Compare with Haskell type signatures.

## Exercise 2: Generic Functions (Medium)
Implement generic swap, first, last functions in all three languages.

## Exercise 3: Algebraic Data Types (Medium-Hard, Haskell)
Create a `Result<T, E>` type in Haskell. Implement `map`, `flatMap`, `unwrap`.

## Exercise 4: Type Inference (Easy)
Write functions letting compiler infer types. Compare C++ auto vs Haskell.

## Exercise 5: Optional Types (Medium)
Implement safe division returning Maybe/Optional in all three languages.
