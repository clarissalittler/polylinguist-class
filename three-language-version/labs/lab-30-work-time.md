### Getting Unstuck

If you're stuck, try:

1. **Rubber duck debugging** - Explain the problem out loud
2. **Simplify** - Can you solve a simpler version?
3. **Search** - Check documentation, Stack Overflow
4. **Ask** - Instructor, classmate, or online community
5. **Take a break** - Sometimes stepping away helps

### Common Issues Checklist

- [ ] Import/module errors - Check file paths and names
- [ ] Type errors - Print values to check types
- [ ] Logic errors - Add print statements to trace execution
- [ ] Off-by-one errors - Check loop bounds carefully
- [ ] Null/None errors - Check for empty cases

## Part 3: Testing Your Code (10 minutes)

### Write At Least One Test

Even a simple test is better than none:

```python
# Python example
def test_basic_functionality():
    result = your_function(input)
    assert result == expected_output

# Run with: pytest test_file.py
```

```rust
// Rust example
#[test]
fn test_basic() {
    assert_eq!(your_function(input), expected);
}

// Run with: cargo test
```

### Test Ideas for Common Projects

**Game:** Test that commands work correctly
**Tool:** Test with known input/output pairs
**Parser:** Test valid and invalid inputs
**Data project:** Test calculations on sample data
